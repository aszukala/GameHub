This website is called GameHub. It is an accumulation of different games that the user can play against the computer. On this site, you can play Rock Paper Scissors, a Number
Guessing game, and a Memory game. Before the user is brought to the game page, they are required to submit their name. They are then brought to the game page where it says
"Welcome to the game page [name]!" For the Rock Paper Scissors game, the computer has a random choice in a array of "rock" "paper" or "scissors". Paper beats rock, rock beats 
scissors, and scissors beats paper. The user chooses one of those options and the website keeps score of whoever wins each round at the bottom. For the Number Guessing game, the 
computer has a random number picked from 1-100. The user has to guess which number the computer picks out. As the user guesses, the computer either says "too high" or "too low."
Using these hints, the user should be able to guess the number it 7 guesses or under. Finally, for the Memory game, the user has a row of blank boxes that they can click on to 
flip and reveal a cat picture. Then, the user must remember the location of the picture and try to fond the matching one. Once all of them are found, the user wins. My team member
was Caitlin Justice. She worked on opening and naming the files, as well as creating the Memory and Number Guessing games. I created the Rock Paper Scissors game, as well as the 
homepage that implememnts user input (their name).
